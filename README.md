# Getting Started

  Send spreadsheet to database with NodeJS.

## Installation

```sh
yarn install
```
## Usage

```sh
yarn dev
```
```sh
routes:
  POST http://localhost:3535/files {
      Body: Multipart Form
                name: file
  }
```
